# Olist Sales Analysis Dashboard

## Overview

This project presents an end-to-end sales analytics solution built in Power BI using the Olist E-commerce dataset. The dashboard provides actionable insights into revenue performance, customer behavior, product performance, delivery metrics, and business opportunities.

## Business Objectives

- Analyze overall sales performance
- Identify top-performing products and categories
- Understand customer distribution and retention
- Track delivery performance and operational risks
- Generate business-driven insights for decision making

## Tools & Technologies

- Power BI
- DAX (Data Analysis Expressions)
- Power Query
- Data Modeling
- Data Visualization
